🎮 使用指南

在主应用中使用代码高亮功能：

  1. 切换显示模式： 在主界面的"显示模式"下拉框中选择"带注释的代码"
  2. 选择算法： 选择任意一种排序算法（支持全部8种）
  3. 开始排序： 点击"开始排序"按钮
  4. 观察高亮： 在排序过程中，右侧代码区域会实时高亮显示当前执行的代码行
  5. 语言切换： 切换中英文时，代码注释和高亮会同步更新
  6. 教学模式： 在教学模式下，每一步都会详细显示正在执行的代码
